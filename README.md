# E-COMMERCE-DASHBOARD-POWERBI-

Designed and developed an interactive Power BI dashboard to analyze e-commerce sales performance across categories, regions, time periods, and customer payment preferences with dynamic filtering.

1.OVERVIEW

This Project helps us to focus on transforming raw data into a meaningful and interactive dashboard.The dashboardenables business users to monitor key performance indicators,analyze sales and profit trends,also helps us to understand customer behaviour and evaluate regional performance through slicers and visuals. The snapshot shown represents Karnataka state, and all visuals dynamically update when the state or quarter selection is changed.

2.PROBLEM STATEMENT AND OBJECTIVE

3.DATASET OVERVIEW

- It's a Transactional E-commerce data.
- There are two data tables: Orders and Order Details
- The dataset includes order dates, state and city information, product categories and sub-categories, quantity sold, revenue, profit, and payment modes.
  
4.TECH STACK

The dashboardwas built using the following tools and technologies:
- Power BI Desktop - Main data visualization platform used for dashboard creation.
- Power Query - Used for transforming,cleaning and preparing it for further analysis.
- DAX(Data Analysis Expressions) - Used for calculated measures,KPI's and dynamic visuals.
- Data Modelling - Relstionships between Orders and Details table.
- File format - .pbix for development and .png for dashboard preview.
  
5.KEY INSIGHTS

NOTE: The following insights are derived for the selected state [KARNATAKA].All insights will change when a different state and quarter is selected using slicer.

- Clothing emerged as the highest-selling category by quantity.
- Accessories and Printers are identified as top-performing sub-categories.
- Gift Cards and UPI are the most preferred payment modes.
- Bangalore was identified as the top revenue-generating city within Karnataka.
- Dynamic slicers allow instant comparison of performances across different states and quarters.
